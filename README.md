# New Project

This repository contains common design decisions for web app projects.

* Copy the docs dir to a new project, and tell the agent to reference `./docs/project-standards/`
* Write a markdown or text file with the description of how you want your new app to work
* Point the agent at your project description and the project-standards dir, and tell it to create a phased approach for implementing your app in ./docs/projects/init/
* Review the project plan and make corrections
* In a loop, tell the agent to execute the next phase of the plan, and review/commit between phases
